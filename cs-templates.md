```
3. 아래 정보를 알려주시면 교환 처리 도와드리겠습니다:
   - 주문번호:

   - 교환 희망 상품:

빠르게 처리해 드리겠습니다. 감사합니다.
```

### 반품/환불 요청
```
안녕하세요, [상호명]입니다.
불편을 드려 죄송합니다.

반품/환불 요청 주셨군요. 아래 내용 확인 부탁드립니다.

✅ 반품 가능 조건
- 상품 수령 후 7일 이내
- 미개봉/미사용 상태
- 상품 하자 또는 오배송의 경우 왕복 배송비 무료

❌ 반품 불가 조건
- 개봉 후 사용한 상품
- 상품 훼손 또는 분실

주문번호와 반품 사유를 알려주시면 바로 안내해 드리겠습니다. 감사합니다.
```

## 4. 결제 관련

### 결제 오류 문의
```
안녕하세요, [상호명]입니다.
불편을 드려 죄송합니다.

결제 관련 오류가 발생하셨군요.
아래 방법을 먼저 시도해보시겠어요?

1. 브라우저 새로고침 후 재시도
2. 다른 결제 수단으로 변경 후 시도
3. 다른 브라우저(크롬 권장)에서 시도

위 방법으로도 해결이 안 되시면 주문번호 또는 오류 화면 캡처를 보내주시면 빠르게 확인해 드리겠습니다. 감사합니다.
```

### 환불 처리 문의
```
안녕하세요, [상호명]입니다.

환불 처리 현황을 안내해 드리겠습니다.

반품 상품 입고 확인 후 영업일 기준 3~5일 이내에 환불 처리됩니다.
카드 결제의 경우 카드사 정책에 따라 취소까지 3~7일이 소요될 수 있습니다.

주문번호를 알려주시면 정확한 처리 현황을 확인해 드리겠습니다. 감사합니다.
```
